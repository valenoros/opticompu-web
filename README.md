Opticompu | High-Performance Optimization
Este es el repositorio oficial de la landing page de Opticompu, un servicio especializado en la optimización de hardware y software para gaming de alto rendimiento.

🚀 El Proyecto
La web fue concebida como una plataforma directa y minimalista para conectar con usuarios que buscan maximizar sus FPS y reducir la latencia en juegos competitivos como CS2, Valorant y Rocket League.

Tecnologías Utilizadas
HTML5 & CSS3: Estructura y diseño desde cero.

Google Fonts (Montserrat): Para una tipografía moderna y técnica.

IA Assisted Development: El código fue desarrollado y refinado utilizando la IA de Visual Studio Code, permitiendo un despliegue rápido y una arquitectura limpia.

Netlify: Hosting y despliegue continuo.

🛠️ Características Técnicas
Diseño Responsivo: Adaptado para una visualización perfecta en dispositivos móviles y escritorio.

Arquitectura Stealth: Interfaz en negro absoluto con acentos en blanco para reflejar una identidad de marca técnica y sobria.

Animaciones de Entrada: Implementación de keyframes personalizados para mejorar la experiencia de usuario al cargar la página.

🧑‍💻 Sobre el Desarrollador
